3- Determinar si un número es primo o no

Inicio 
	Escribe "Elija un número"
	Lee "numero"
	Si numero == |1| o numero == |2|
		Escribe "Sí es primo" 
	Si numero % 2 == 0 
		Escribe "No es primo"
	Para (i = 3, i < numero / 2, i + 2)
		Si numero % i == 0
			Escribe "No es primo"
	Si ninguna de las anteriores condiciones se cumple 
		Escribe "Sí es primo"
	
Fin
